دليل تفعيل خدمات الـ OTP (SMS والبريد الإلكتروني)

يوضح هذا المستند دليلاً تفصيلياً للمطورين لشرح كيفية تفعيل خدمة رموز التحقق (OTP) حقيقةً عبر الرسائل النصية القصيرة (SMS) والبريد الإلكتروني (Email) في خادم المشروع.

أولاً: دليل الاشتراك في خدمات الـ SMS OTP (الرسائل النصية القصيرة)

لكي تصل رموز التحقق إلى جوالات قيّمي المساجد حقيقةً، يجب ربط السيرفر بمزوّد خدمة رسائل (SMS Gateway).

1. المتطلبات والاشتراك:

يتطلب الربط بالبوابات المحلية (مثل الشركات السورية كـ Syriatel/Al-Amin) أو العالمية (مثل Twilio/Vonage) ما يلي:

  • حساب مطور (Developer Account): فتح حساب في موقع الشركة المزودة للخدمة.
  • رصيد رسائل (SMS Credits): شحن الحساب بمبلغ مالي أو حزمة رسائل لدعم الإرسال.
  • معرف المرسل (Sender ID): الاسم التجاري المعتمد الذي ستصل الرسائل باسمه للقيمين (مثل: AwqafDamas).
  • مفاتيح الوصول (API Keys): رموز السرية التي يمنحها الموقع للسيرفر ليسمح له بالإرسال البرمجي.

2. طريقة الدمج في الكود (server.js):

عند طلب الرمز، يتم إرسال طلب HTTP POST لمزود الخدمة. كمثال للربط مع بوابة رسائل:

const https = require('https');

function sendSMS(phoneNumber, code) {
  const message = `رمز التحقق الخاص بك لتطبيق الأوقاف هو: ${code}`;
  
  const data = JSON.stringify({
    apiKey: 'YOUR_SMS_API_KEY',
    sender: 'AwqafDamas',
    to: phoneNumber,
    message: message
  });

  const options = {
    hostname: 'api.smsprovider.com', // رابط API الخاص بالشركة
    path: '/v1/send',
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'Content-Length': data.length
    }
  };

  const req = https.request(options, (res) => {
    res.on('data', (d) => {
      console.log(`[رسائل] رد بوابة الـ SMS: ${d.toString()}`);
    });
  });

  req.on('error', (error) => {
    console.error(`[أمن] فشل إرسال رسالة الـ SMS:`, error);
  });

  req.write(data);
  req.end();
}

ثانياً: دليل تفعيل خدمة الـ OTP عبر البريد الإلكتروني (Email OTP)

يمكن للسيرفر إرسال رمز التحقق كرسالة بريد إلكتروني إلى البريد الرسمي للقيم (مثل: ahmad@awqaf-damas.gov.sy).

1. المتطلبات:

  • حساب بريد إلكتروني رسمي (SMTP Server): مثل حساب الوزارة الرسمي، أو حساب Gmail مخصص للتطبيق.
  • إعدادات منفذ الإرسال (Host / Port / Secure): عنوان خادم البريد ومنفذ الاتصال (مثل 465 للـ SSL).

2. كيفية إضافة الكود في السيرفر (server.js):

نستخدم مكتبة nodemailer المصممة لإرسال رسائل البريد الإلكتروني في Node.js.

الخطوة 1: تثبيت المكتبة في مجلد السيرفر: 
npm install nodemailer
الخطوة 2: إضافة كود الإرسال في ملف server.js: 
const nodemailer = require('nodemailer');

// 1. إعداد ناقل البريد الإلكتروني (Transporter)
const mailTransporter = nodemailer.createTransport({
  host: 'smtp.gmail.com',         // أو خادم بريد الوزارة
  port: 465,
  secure: true,
  auth: {
    user: 'your-app-email@gmail.com', // بريد الإرسال
    pass: 'your-app-password'         // كلمة مرور التطبيقات للحساب
  }
});

// 2. دالة إرسال بريد الـ OTP
function sendEmailOTP(targetEmail, code) {
  const mailOptions = {
    from: '"مديرية أوقاف دمشق" <your-app-email@gmail.com>',
    to: targetEmail,
    subject: 'رمز التحقق — تطبيق تبادل مستلزمات المساجد',
    html: `
      <div style="direction: rtl; text-align: right; font-family: Tahoma, sans-serif; padding: 20px; border: 1px solid #ccc; border-radius: 5px;">
        <h2 style="color: #006643;">مديرية أوقاف دمشق</h2>
        <p>مرحباً بك، لقد طلبت رمز الدخول لتطبيق تبادل مستلزمات المساجد.</p>
        <div style="font-size: 24px; font-weight: bold; background-color: #f5f5f5; padding: 10px; text-align: center; border-radius: 5px; letter-spacing: 5px; color: #006643;">
          \${code}
        </div>
        <p style="color: #666; font-size: 12px; margin-top: 20px;">هذا الرمز صالح للاستخدام لمرة واحدة فقط وصلاحيته محدودة.</p>
      </div>
    `
  };

  mailTransporter.sendMail(mailOptions, (error, info) => {
    if (error) {
      return console.error('[بريد] فشل إرسال رمز التحقق:', error);
    }
    console.log('[بريد] تم إرسال الرمز بنجاح. معرف الرسالة: %s', info.messageId);
  });
}
الخطوة 3: استدعاء الدالة داخل مسار المصادقة في السيرفر (server.js):

في السيرفر بدلاً من طباعة الرمز، نقوم بتوجيه الإرسال حسب المدخلات:

// استدعاء الإرسال الفعلي في وضع الإنتاج
if (PROD) {
  if (user.email) {
    sendEmailOTP(user.email, code); // إرسال عبر البريد الإلكتروني
  }
  if (user.phone) {
    sendSMS(user.phone, code);      // إرسال عبر رسائل الـ SMS
  }
}

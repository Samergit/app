مديرية أوقاف دمشق

هذا الفهرس يربط كل ملفات التوثيق. النظام يتكوّن من ثلاث نسخ من واجهة واحدة (ويب + أندرويد + آيفون) متصلة بسيرفر مشترك.

أولاً: التوثيق البرمجي (للمطوّرين)

# الملف المحتوى
01 01_توثيق_الخادم.md بنية السيرفر الخلفي، شرح كل وحدة وكل دالة، الإعداد والأمان
02 02_مرجع_API.md مرجع كامل لكل نقاط الـ API: الطلب، الرد، الأخطاء، أمثلة
03 03_قاعدة_البيانات.md نموذج البيانات، الكيانات، الحقول، آلة الحالات
04 04_توثيق_تطبيق_الموبايل.md بنية مشروع Flutter، شرح كل ملف، البناء لأندرويد وآيفون

ثانياً: توثيق المحتوى وأدلة المستخدم

# الملف المحتوى
05 05_دليل_المستخدم_الويب.md دليل استخدام النسخة الويب خطوة بخطوة لكل دور
06 06_دليل_المستخدم_الموبايل.md دليل استخدام تطبيق الهاتف (أندرويد/آيفون)
07 07_توثيق_المحتوى_والشاشات.md توثيق كل شاشة وحقل وزر ورسالة في كل النسخ
